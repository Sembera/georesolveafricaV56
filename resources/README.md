## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**: Follow the aspect ratios of placeholder images
- **Naming**: Use descriptive filenames
- **Quality**: High quality but web-optimized

## Contentful Integration

When using Contentful CMS:
1. Upload images to Contentful Media Library
2. Reference images in your content entries
3. The website will automatically use Contentful URLs
4. Local images in this directory serve as fallbacks

## Image Optimization

For best performance:
- Use modern formats (WebP) when possible
- Optimize images before uploading
- Consider using responsive images with multiple sizes
- Use lazy loading for better page performance

## Accessibility

- Always include descriptive alt text
- Ensure sufficient color contrast in images
- Provide text alternatives for complex diagrams
